# このゲームについて
現在、鋭意制作中のオリジナル人狼ゲームです。  
ひとしずくPさんとやま△さんの楽曲「Mr.シャーデンフロイデ」に強いインスピレーションを受けています。
# コンセプト
従来の人狼ゲームより、遅いゲームテンポでじっくりと強い疑心暗鬼に苛まれながらプレイするというコンセプトを持っています。
# ゲームの流れ
大まかな流れとしては従来の人狼ゲームと殆ど変わりません。
しかし、特定の状況下では自分たちの判断でゲームを終わらせる事ができます。
# 特色
このゲームでは「人狼」という役職はなく、役職とは別に「悪魔憑き」という属性が与えられ人狼扱いとされます。
死亡時に死因が付与されるため、ある程度の推理の助けとなります。
また、人狼陣営（以下、悪魔憑き）にはフロイデ値、村人陣営（非悪魔憑き）にはシャーデン値が付与されます。
### フロイデ値
毎ターン減少し、他人の死亡、役職の固有行動で増加します。  
フロイデ値が一定以上になると次の昼のターンで発狂します。  
発狂したタイミングで悪魔憑きより非悪魔憑きのほうが多いと取り押さえられ悪魔憑きの敗北となります。  
増減値はある程度の範囲のランダムで決まります。  
### シャーデン値
毎ターン増加し、他人の死亡、被投票、夜に一人でいるなどの条件でも追加で増加します。    
役職の固有行動、夜のターンに他人と会話、会食するなどの行動で減少させることができます。  
一定以上になるとランダムな方法で自死します。人数が減るだけでなく、悪魔憑きとの殺害と混同することで会議のノイズになります。
フロイデ値と同じく増減値はある程度の範囲のランダムで決まります。  
# 勝利条件
悪魔憑き：他人全員を殺害すると勝利します。  
狂人：ゲーム終了時、悪魔憑きによる最後の殺害の対象が自分であると勝利します。   
非悪魔憑き：ゲーム終了時、悪魔憑きが追放されているかつ自分が生存していると勝利します。  
悪魔憑きと狂人、狂人と非悪魔憑きは同時に勝利条件を達成する場合がありますが、その場合は両陣営勝利となります。  
このゲームでは非悪魔憑きも狂人も最終的に生存していないと勝利とならないため、人狼ゲームによくある「とりあえず村人を吊る」
であったり「ゲームを続けるために犠牲になる」という選択を取りにくくしています。  
全員が生に貪欲になることで悪魔憑きが隠れやすく後述の自分たちの判断でゲームを終了させる判断も鈍らせる目的があります。
# 現在予定している役職
### 村人
非悪魔憑き：~~特に決まった能力のない村人です。~~ 夜のターン訪問を受け入れるか拒否するかを選択できます。    
悪魔憑き：誰か一人を狂人にすることができます。後述する神官は狂人にならないためただ回数を消費します。  
補足：特に決まった能力はありませんが、村人なので他人の家に行くことに違和感を持たれず、運が良ければ狂人という味方を増やすこともできます。  
### 狂人
非悪魔憑き：狂人になる前の役職に依存します。悪魔憑きが誰かを知っています。  
悪魔憑き；ゲーム開始後に付与される役職なので悪魔憑きになる可能性はありません。  
補足：基本的に最後まで生存している必要があるため、場を混乱させるだけでは成り立たないようにしています。  
### 騎士
非悪魔憑き：基本的な人狼と同じです。  
悪魔憑き：誰かを守ることでフロイデ値を減少させることができます。  
### 葬儀屋
非悪魔憑き：他人の死亡でシャーデン値が増加しません。昼のターンで死亡した一人を指定しエンバーミングを行い、そのプレイヤーの役職と死因と悪魔憑きか否かがわかります。  
悪魔憑き：エンバーミングでフロイデ値を回復できます。  
補足：基本的な人狼における霊媒師の役回りです。死因がわかるのが葬儀屋のみであるため偽装も可能です。また、悪魔憑きかどうか分かる能力も後述の自分たちの判断でゲームを終了させるタイミングでミスリードを誘えると考えています。  
### 医者
非悪魔憑き：夜に調合を行うことで自分のシャーデン値を回復しながら、調合した薬で他人のシャーデン値を下げることができます。また、訪問したプレイヤーがそのターンに自死を選んだときは薬を使用することで確率で蘇生できます。  
悪魔憑き：毒を使用することで投与した次の夜のターンで殺害できます。この時死因は服毒死になります。薬を使用するとフロイデ値を減少させることもできます。  
補足：蘇生に関してはかなりロマン技に近いものになると考えています。今のところ唯一の遅延付きの殺害持ちなので同時に二人殺害することも可能です。もちろん二倍でフロイデ値も増えますが。  
### 神官
非悪魔憑き：夜のターン誰か一人に祝福を与えることで対象のストレス値を減少させ、フロイデ値を増加させることができます。  
このキャラクターが生存している時、全プレイヤーは神託を受けることができます。神託では、悪魔憑きによる殺害が行われたかとその対象、そして悪魔憑きが村から追放されたかどうかがわかります。  
このキャラクターの死亡以降は神託を受ける事ができないため、生存プレイヤーが悪魔憑きを追放したと判断した時点でゲームを終わらせる必要があります。  
ゲームを終わらせた時点で悪魔憑きが生存していた場合悪魔憑きによる虐殺が行われ悪魔憑きの勝利となります。この時狂人が生存していたら狂人も勝利となります。
悪魔憑き：このキャラクターは悪魔憑きになりません。  
補足：先程からの「ゲームを自分たちの判断で終了させる」状況はこのキャラクターの死亡以降となります。このキャラクターが生存しているかどうかでゲーム性がガラリ変わるため、悪魔憑きとしても非悪魔憑きとして鍵となる役職です。
### 今後について
ひとまず、現在の想定での制作を終わらせたあとに以下の課題解決と調整を行う予定です。  

1. シャーデン値フロイデ値の増減量、上限量の設定
2. 各役職のパワーバランス
   - 各固有行動の見直し
   - フロイデ値シャーデン値の増減量とタイミング
3. 役職、陣営の追加
4. UIのデザイン
