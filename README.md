# このゲームについて
現在、鋭意制作中のオリジナル人狼ゲームです。  
ひとしずくPさんとやま△さんの楽曲「Mr.シャーデンフロイデ」に強いインスピレーションを受けています。
# コンセプト
従来の人狼ゲームより、遅いゲームテンポでじっくりと強い疑心暗鬼に苛まれながらプレイするというコンセプトを持っています。
# ゲームの流れ
大まかな流れとしては従来の人狼ゲームと殆ど変わりません。
しかし、特定の状況下では自分たちの判断でゲームを終わらせる事ができます。
# 特色
このゲームでは「人狼」という役職はなく、役職とは別に「悪魔憑き」という属性が与えられ人狼扱いとされます。  
また、人狼陣営（以下、悪魔憑き）にはフロイデ値、村人陣営（非悪魔憑き）にはシャーデン値が付与されます。
### フロイデ値
毎ターン減少し、他人の死亡、役職の固有行動で増加します。  
フロイデ値が一定以上になると次の昼のターンで発狂します。  
発狂したタイミングで悪魔憑きより非悪魔憑きのほうが多いと取り押さえられ悪魔憑きの敗北となります。
### シャーデン値
毎ターン増加し、他人の死亡、被投票、夜に一人でいるなどの条件でも追加で増加します。    
役職の固有行動、夜のターンに他人と会話、会食するなどの行動で減少させることができます。  
一定以上になるとランダムな方法で自死します。人数が減るだけでなく、悪魔憑きとの殺害と混同することで会議のノイズになります。
